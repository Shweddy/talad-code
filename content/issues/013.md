สวัสดีชาวตลาดทุกคน! ฉบับที่ 13 มาในธีม **ทำเอง ใช้เอง ไม่ต้องง้อใคร** —
เดือนนี้แผงส่วนใหญ่เป็นของที่ยกไปตั้งบนเซิร์ฟเวอร์ตัวเองได้ ไม่ต้องพึ่งบริการ
เสียเงินรายเดือนของเจ้าไหน

เดินเข้ามาแผงแรกจะเจอ **Ollama** กับ **Thonburian Whisper** สองแผง AI
ที่รันบนเครื่องตัวเองได้เลยไม่ง้อ API ตัวหลังเป็นฝีมือทีมวิจัยไทย
(Biodatlab มหิดล ร่วมกับ Looloo Technology) ที่ถอดเสียงพูดภาษาไทยได้แม่น
เป็นพิเศษ — หยิบใส่ตะกร้าได้ไม่อาย ผลงานคนไทยของจริง

ถัดมาเป็นแผงของคนอยากประหยัดค่า hosting: **PocketBase** อัดแบ็กเอนด์ทั้งชุด
ไว้ในไฟล์เดียว, **Coolify** ให้ดีพลอยเองแทน Heroku/Vercel, และ **Immich**
เก็บรูปครอบครัวไว้เองไม่ต้องฝากคลาวด์ใคร

ปิดท้ายด้วยของใช้ประจำวันอย่าง **Biome** ที่มาแทน Prettier กับ ESLint
สองตัวในทีเดียว, **Expo** สำหรับใครอยากลองทำแอปมือถือด้วย React ที่มีอยู่แล้ว
และ **Godot** เกมเอนจินฟรีสำหรับใครอยากพักจากงานสาย web ไปทำเกมเป็นงานอดิเรก

ขอบคุณอาสาสมัครทุกคนที่ช่วยกันคัดของมาลงแผงเดือนนี้ ถ้าเจอโปรเจกต์เด็ด
